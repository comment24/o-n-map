# 온누리상품권 가맹점 리스트 조회 방법 리서치

## 개요
온누리상품권은 중소벤처기업부에서 전통시장 및 소상공인을 지원하기 위해 발행하는 상품권으로, 전국 약 13만개 이상의 가맹점에서 사용 가능합니다.

## 1. 공식 웹사이트를 통한 조회

### 주요 공식 사이트
- **온누리상품권 가맹점 찾기**: https://onnurishopfinder.or.kr/
- **온누리상품권 공식 홈페이지**: https://onnurigift.or.kr/
- **가맹점 관리 시스템**: https://ongift.or.kr/
- **소상공인시장진흥공단**: https://www.sbiz.or.kr/

### 웹 검색 방법
1. 온누리상품권 공식 홈페이지 접속
2. 상단 메뉴 - 사용처 - 가맹점 찾기 선택
3. 지역별/카테고리별 검색 가능
4. 지도 기반 가맹점 검색 서비스 제공

## 2. 공공데이터포털을 통한 데이터 접근

### 파일 데이터 다운로드 (CSV)

**데이터 위치**
- URL: https://www.data.go.kr/data/3060079/fileData.do
- 제공기관: 소상공인시장진흥공단
- 최신 데이터: 전국 온누리상품권 가맹점 현황_20240731

**데이터 필드**
- 가맹점명 (Merchant Name)
- 소속 시장명 (Market Name)
- 소재지 (Address/Location)
- 취급 품목 (Product Categories)
- 충전식카드 취급 여부 (Rechargeable Card Support)
- 지류 취급 여부 (Paper Voucher Support)
- 모바일 취급 여부 (Mobile Voucher Support)

**다운로드 방법**
- 로그인 없이 파일 데이터 다운로드 가능
- CSV 형식으로 제공

### Open API 활용

**API 형식**
- RestAPI 기반
- JSON/XML 형식 지원
- 공공데이터 활용지원센터에서 파일 데이터를 자동으로 Open API로 변환

**API 신청 절차**
1. 공공데이터포털(www.data.go.kr) 회원가입
2. 로그인 후 API 인증키 신청
3. 개발계정 신청 페이지에서 정보 입력
4. 인증키 발급 후 API 활용 가능

**참고 데이터셋**
- 전국 온누리상품권 가맹점 현황: www.data.go.kr/data/3060079/fileData.do
- 모바일상품권 가맹점: www.data.go.kr/data/15090978/fileData.do
- 메타데이터: www.data.go.kr/dcat/metadata/15091186

## 5. 가맹점 데이터 활용 시 주의사항

1. **정기 갱신**: 가맹점은 매년 갱신되므로 최신 데이터 확인 필요
2. **데이터 형식**: CSV, JSON, XML 형식으로 제공
3. **접근성**: 파일 데이터는 로그인 없이 다운로드 가능하나, API는 인증키 필요
4. **업데이트 주기**: 공공데이터포털의 데이터 업데이트 일자 확인 필요

## 6. 개발자를 위한 정보

### 데이터 활용 방법

**방법 1: 파일 다운로드**
```
1. https://www.data.go.kr/data/3060079/fileData.do 접속
2. CSV 파일 다운로드
3. 로컬에서 데이터 파싱 및 활용
```

**방법 2: Open API 활용**
```
1. 공공데이터포털 회원가입
2. 온누리상품권 가맹점 API 신청
3. 인증키 발급
4. REST API 호출 (JSON/XML)
```

### 지도 기반 서비스
- 소상공인시장진흥공단에서 '지도 기반 가맹점 검색' 서비스 전면 도입
- 온누리상품권 홈페이지에서 지도 기반 검색 가능

## 결론

온누리상품권 가맹점 리스트는 다음 세 가지 주요 방법으로 접근 가능합니다:

1. **일반 사용자**: 공식 웹사이트(onnurishopfinder.or.kr) 또는 모바일 앱 활용
2. **데이터 분석가**: 공공데이터포털에서 CSV 파일 다운로드
3. **개발자**: 공공데이터포털 Open API 활용 (인증키 필요)

가장 간편한 방법은 공식 웹사이트나 모바일 앱을 사용하는 것이며, 대량의 데이터를 처리하거나 자동화가 필요한 경우 공공데이터포털의 파일 또는 API를 활용하는 것이 효율적입니다.
